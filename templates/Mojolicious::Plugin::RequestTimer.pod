=head1 名前

Mojolicious::Plugin::RequestTimer - リクエストタイマープラグイン

=head1 使い方

  # Mojolicious
  $self->plugin('RequestTimer');

  # Mojolicious::Lite
  plugin 'RequestTimer';

=head1 説明

[[Mojolicious::Plugin::RequestTimer]]は、リクエスト時の情報を収集して記録するためのプラグインです。
これはコアプラグインなので、いつでも使用可能であり、新しいプラグインを作成する方法を学ぶための良いコード例になります。

=head1 メソッド

[[Mojolicious::Plugin::RequestTimer]]は、[[Mojolicious::Plugin]]のすべてのメソッドを継承し、以下の新しいメソッドを実装しています。

=head2 register

  $plugin->register;

[[Mojolicious]]アプリケーションにプラグインのフックを登録します。
